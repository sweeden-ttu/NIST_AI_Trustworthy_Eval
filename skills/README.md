# Template skills (Cursor)

Copy or symlink a skill folder into your Cursor skills directory so agents discover it, for example:

```bash
ln -s "$(pwd)/homework-to-ieee-article" ~/.cursor/skills/homework-to-ieee-article
ln -s "$(pwd)/rd-agent-mcp-operator" ~/.cursor/skills/rd-agent-mcp-operator
```

Authoring format: follow Cursor’s SKILL.md conventions (YAML frontmatter + sections).
