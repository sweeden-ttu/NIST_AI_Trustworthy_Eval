\section*{Ethics Statement}

This work uses only educational cryptography artifacts and public TLS endpoints. The toy AES implementation must not be presented as side-channel resistant or standards compliant for production. ECB/CBC demonstrations intentionally highlight leakage patterns; readers should not infer that header-spliced BMP previews are security mechanisms. OpenSSL key/IV material emitted for reproducibility may be sensitive if committed to public repositories---redact or exclude per institutional policy. TLS probing should respect site terms and local law; the bundled host list is illustrative and may be replaced.
